個人情報の定義

本プライバシーポリシーが対象とする「個人情報」の定義は、日本の「個人情報の保護に関する法律」（平成 15年法律第57号。その後の改正及び変更を含み、以下「個人情報保護法」といいます。）をはじめとする適用ある国又は地域のプライバシー関係法令（以下 「法令等」といいます。）の定めるところに従います。

なお、日本の個人情報保護法の「個人情報」は、生存する個人に関する情報で、①その情報に含まれる氏名、 生年月日その他の記述等により、特定の個人を識別することができるもの（その情報自体によって特定の個人を識別できるもののほか、他の情報と容易に照合す ることができ、それによって特定の個人が識別できるものも含みます。）及び②指紋、声紋など個人情報保護法により、「個人識別符号」として定められている ものをいいます。

また、本プライバシーポリシーが日本に所在するお客様の個人情報に適用される場合、「個人情報」には、個人情報保護法の定める「個人データ」及び「保有個人データ」を含むものとします。

  1. 1.

  2. 2.

  4. 4.
    個人情報の不正なアクセス、漏えい、改ざん、滅失、き損等を防止する等安全管理のために必要かつ適切な措置を講じるよう努めます。
  5. 5.

個人情報の管理

ポケモングループでは、お客様から収集した個人情報の漏えい、滅失、毀損等を防止するために、以下に掲げる措置をはじめとした個人情報の安全管理のための必要かつ適切な措置を講じるとともに、継続的な見直しや改善に取り組むよう努めるものとします。

  1. 1.
    組織的安全管理措置
    情報管理責任者の選任、個人情報の取扱いに関する専門部署の設置、漏えい等の事案発生時における対応フローその他の組織体制の整備を行います。また、個人情報の取扱いに関する社内規程等を定め、適切に運用します。
  2. 2.
    人的安全管理措置
    個人情報の取扱いに関する情報を周知徹底し、個人情報の安全管理のための従業員に対する適切な監督や教育を行います。
  3. 3.
    物理的安全管理措置
    個人情報を取り扱う区画を管理し、当該区画外での個人情報の取扱いの制限等を行い、個人情報を取り扱う機器、電子媒体等の盗難や漏えいを防止します。
  4. 4.
    技術的安全管理措置
    個人情報を取り扱うことのできる情報システム、及び当該情報システムを使用できる従業者を限定し、適切なアクセス制御を行います。また、外部からの不正アクセスや不正ソフトウェアを防止する仕組みを導入し、適切に運用します。
  5. 5.
    外的環境の把握
    外国において個人情報を取り扱う場合、当該外国の個人情報の保護に関する制度等を把握した上で、個人情報の安全管理のために必要かつ適切な措置を講じます。
